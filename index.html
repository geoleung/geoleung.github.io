About Me

I'm Geoffrey, a web developer passionate about writing organized code. I have found much enjoyment and excitement from coding with JavaScript and React, and my dream is to create web applications that will improve people's lives. I work well in fast-paced environments and I am a quick and adaptable learner. Learning new coding languages and technologies is always interesting to me, as I am constantly striving to improve myself.
